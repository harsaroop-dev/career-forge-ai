An AI-powered career dashboard that analyzes Resumes against Job Descriptions.

## Tech Stack

- **Frontend**: Next.js, Tailwind CSS, Lucide React, Axios
- **Backend**: FastAPI, Python, Groq (Llama 3.3 70B)
- **Database**: Supabase (Vector Store / pgvector)
- **AI/ML**: LangChain, HuggingFace Embeddings

## Setup

1. Clone the repo.
2. Setup **Backend**: Install `requirements.txt` in a `venv`.
3. Setup **Frontend**: Run `npm install`.
4. Add your `.env` variables.
